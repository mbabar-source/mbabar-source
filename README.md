🚀 Featured Project — ContentFlow AI

ContentFlow AI is an AI-powered content idea generation platform designed for content creators, influencers, freelancers, and small businesses.

The application allows a user to enter a topic and then:

* Collects relevant source content
* Stores sources in a SQLite database
* Retrieves relevant information using RAG
* Uses AI to analyze the available context
* Generates structured content ideas
* Saves generated results for future use
* Displays the output through a professional frontend
