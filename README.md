## 🚀 Featured Project — ContentFlow AI

**ContentFlow AI** is an AI-powered content idea generation platform designed for content creators, influencers, freelancers, and small businesses.

The application follows this core workflow:

`Enter Topic → Collect Sources → Store Data → Retrieve with RAG → Analyze with AI → Generate Content Ideas`

### What it does

- Collects relevant source content for a user-provided topic
- Stores collected sources in a SQLite database
- Retrieves relevant saved information using RAG
- Uses AI to analyze the available context
- Generates structured content ideas
- Saves generated results for later use
- Displays the final results through a professional frontend
